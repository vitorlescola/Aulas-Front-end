Linguagem de programação C

Essa pagina apresenta um resumo basico do conteudo estudado sobre a linguagem C, com uma avaliação ao final.

Introdução

C é uma linguagem de programação de baixo nivel, criado por Dennis Ritchie in 1972.

C é considerada uma linguagem fundamental pois ela é a base de varias outras linguas, e é utilizada até hoje por sua agilidade é baixo uso de memoria (clique e segure para mais informações, em todas as fichas).

Sintaxe

#include <nome da biblioteca> //adiciona uma biblioteca

int main(){} //indica a area principal do código

; //indica ao código onde que a linha acaba

return 0; //termina o código

output

printf("");

esse é o comando para dar um output de comando, escrevendo algo na tela, para colocar uma varivale é preciso utilizar:
printf("%d %f %c %s",int,float,char,string);

mais informação sobre na ficha das variaveis

Variaveis

As variaveis são onde você salva uma informação no código

int numeros; //valores inteiros

float reais; //valores reais

char letra; //caracteres

char nome[50]; //textos, com o valor entre [] sendo a quantidade de caracteres

Input

scanf("%d",variavel);

esse é o comando para aceitar input do usuario, e salvar o valor em uma variavel do tipo especificado.

If Else

if(condição){faz isso;
}else{
se não, fassa isso;
}

verifica se a condição entre () é verdadeira, se for executa o código, se não for executa outro código, ou nenhum.

While

while(condição){código}

executa o código enquanto a condição ser verdadeira.

For

for(i=0;<5;i++){código}

precisa de 3 condições, uma variavel de contagem, uma condição de parada, e um encremento (neste caso de 1 a 1 até chegar a 5).

Array

int num[5];

Uma array é uma variavel que salva variaos valores do mesmo tipo.
